Image/Resouce Hosting With GITHUB API

## Prepare

- Create github access token

- reset variable

```sh
gh_repo=resources
gh_host=https://api.github.com
gh_owner=hyongbai

gh_user=hyongbai
gh_emial=hyongbai+github@gmail.com
```

## How to use

```sh
source uploader.sh
GITHUB_TOKEN={your token}
gh_up {file} {remote_path}
```
